Lab File Server

HTTP share for PXE boot assets — ISO images, kernels, initrds and root filesystems.

Download path: /files/  (host directory: ./data/webfs_share)

Boot menus are managed by the iPXE Manager, not from this page. Upload files and build boot entries there — the live menu is served at http://<this-host>:8091/menu.ipxe, and PXE clients are pointed at it automatically by DHCP.

Static iPXE scripts (manual fallback)

  • /ipxe/default.ipxe — chains to the iPXE Manager menu
  • /ipxe/menu.ipxe — static fallback menu
  • /ipxe/linux-kernel-initrd.ipxe — kernel + initrd over HTTP (UEFI-friendly)
  • /ipxe/boot-iso.ipxe — ISO sanboot (BIOS only)
